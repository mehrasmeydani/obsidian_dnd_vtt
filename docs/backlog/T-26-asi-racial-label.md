# T-26 — ASI points mislabeled as racial bonus

**Priority:** P2 · **Size:** XS · **Phase:** 1 · **Depends on:** — · **From:** user testing (todo #mid)

## User story
As a **player** assigning ASI points, I want the ability rows to say the
increase came from an ASI — today everything above the base score is
labeled "racial".

## Acceptance criteria
- [ ] Ability rows show racial and ASI deltas separately ("+2 racial",
      "+1 ASI"), only when non-zero.
- [ ] jsdom test covering a draft with both racial and ASI increases.
