# T-33 — Features grouped by origin

**Priority:** P2 · **Size:** S · **Phase:** 1 · **Depends on:** T-01 · **From:** user testing (todo #minor)

## User story
As a **player**, I want race, class, background, and feat features in
separate groups — class features sorted by level, with feats appearing
both in the feats group and inline in the class-by-level list.

## Acceptance criteria
- [x] Features tile renders grouped: Race / Class (level-sorted, subclass
      included, leveled feats inline) / Background / Feats / Other.
- [x] Feats taken at an ASI level appear in both the Feats group and the
      class progression.
- [x] jsdom test for the grouping.
